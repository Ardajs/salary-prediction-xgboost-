# salary-prediction-xgboost-
Machine learning project that predicts data science salaries using XGBoost regression, feature engineering, and real-world job listing data.

# Salary Prediction with XGBoost

This project predicts Data Science salaries using XGBoost regression.

## Features
- Company info
- Skills (Python, AWS, Spark, Excel, R)
- Job metadata

## Model
XGBoost Regressor

## Performance
R2 Score: ~0.80  
MAE: ~11K
